# 02_4 Dialectica del Modelado (Decision)

## Representacion vs Realidad
- Decision: se acepta el modelo solo si es validable externamente.

## Simplificacion vs Complejidad
- Decision: complejidad solo si aumenta prediccion o explicacion.

## Determinismo vs Estocasticidad
- Decision: se usa estocasticidad cuando mejora ajuste empirico.

## Escalabilidad vs Interpretabilidad
- Decision: si el modelo no es interpretable, se justifica o se rechaza.
